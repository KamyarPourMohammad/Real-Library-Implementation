# Real Library Implementation

##  Introduction
This project is a simple Object-Oriented Python implementation of a library management system.  
It allows an admin to create a library and manage books.

##  Features
- Create a library
- Add books
- Remove books
- List all books
- Count books

##  Project Structure
- `Library` class → manages books
- `Admin` class → manages library creation
- `run_app()` → command-line interface
